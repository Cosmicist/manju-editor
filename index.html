Manju Editor

Manju is a lightweight rich text editor made as a jQuery plugin. It's ideal to use in your projects when you need to enable the user to visually edit content, and, of course, it also allows source code viewing/editing.

Usage

Include the script and styles, and don't forget jQuery!

<link rel="stylesheet" href="css/jquery.manju.css">

<script src="http://cdnjs.cloudflare.com/ajax/libs/jquery/1.8.2/jquery.min.js"></script>
<script src="js/jquery.manju.js"></script>

It is recommended to include the scripts just before closing the body tag.

Now it's as simple as calling manju() on a textarea or any other element that you wish to make editable:

$('.editable').manju();
